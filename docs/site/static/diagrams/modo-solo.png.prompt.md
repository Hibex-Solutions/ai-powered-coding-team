# Diagrama "Modo Solo" — Prompt de geração

A imagem `modo-solo.png` é gerada com **Google Gemini**, mesmo estilo visual da opção **"Ciborgue"** usada no banner do site (ver `.github/assets/banner.png.prompt.md`).

## Prompt utilizado

```
Sci-fi cyberpunk timeline illustration in 16:9 landscape aspect ratio.
Dark deep navy background (#080d1a), subtle hexagonal grid and circuit
lines faintly visible in corners, blue-to-purple neon glow atmosphere,
floating data particles, cinematic lighting, professional sci-fi concept
art style, high detail.

Header label at the top in thin glowing futuristic sans-serif text:
"MODO SOLO".

Composition: a single cyborg character shown at FIVE sequential stages
along a horizontal timeline flowing LEFT → RIGHT. The five figures are
the SAME cyborg (identical face, body, silhouette, proportions), but the
aura glow around each one changes to reflect the role being assumed at
that stage. Between each figure and the next, a vertical glowing strand
of AI neural network (blue-to-purple gradient, particles flowing upward)
receives the delegation and emits a floating holographic artifact panel
whose glow matches the current role color. Thin neural network lines
connect the entire sequence.

Stage breakdown (each figure shows a small floating numbered badge above
it — [1], [2], [3], [4], [5] — and a thin glowing caption beneath):

 [1] Aura EMERALD GREEN (#10b981) — caption "como Engenheiro" —
     delegates "objetivo" → AI column → artifact panel "docs/GOAL.md".
 [2] Aura VIOLET (#7c3aed) — caption "como Analista" —
     delegates "especificação" → AI column → artifact panel
     "docs/BUSINESS.md".
 [3] Aura ELECTRIC BLUE (#4f6ef7) — caption "como Arquiteto" —
     delegates "projeção" → AI column → two stacked artifact panels
     "docs/ARCHITECTURE.md" and "docs/SOLUTION.md".
 [4] Aura PURPLE (#a855f7) — caption "como Designer" —
     delegates "projeção" → AI column → artifact panel
     "docs/GUIDELINE.md".
 [5] Aura EMERALD GREEN (#10b981) — caption "como Engenheiro" —
     delegates "construção" → AI column → glowing terminal panel
     labeled "[SOFTWARE]".

The repeated identical face across all five stages emphasizes that a
single person occupies every role in sequence.

All labels, badges and artifact filenames must be crisp, legible,
futuristic sans-serif. No text outside the specified labels.
```
